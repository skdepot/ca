POS-Point of Sale Warehouse Management Odoo Apps

This odoo apps is used to manage multiple warehouse locations in POS and configuring different warehouse locations in different POS sessions.This Odoo module is show stock of each product in pos screen for the configured warehouse on the POS session, after configure the warehouse for the pos session, pos user can able to see product stock of each inventory stock location for that configured warehouse.

Features

POS Warehouse Configuration.

User can add multiple warehouse from pos back-end..

Print POS Payment Receipt

User can print pos payment receipt with added product quantity on particular location.

Display All Selected Locations on POS Screen.

User can see all selected location on pos screen.

Check Availability

Check Available stock in Other locations from pos screen, also provide notification message if product is out of stock.

Update Quantity on POS Screen.

Update quantity of Product in Selected Locations on POS screen.

Picking Details

Separately create picking entry, user can see picking details in Back-end.

Add Quantity from Different Location for Same Product

User can add quantity from different location for same product.

Warning/Validation Message

When user enter more then available quantity that time raise warning/valition message of product details with stock location.
